Charte d'utilisation

En utilisant le Carnet en Ligne (CEL) et les outils de saisie associés (widgets), vous acceptez que Tela Botanica enregistre et, le cas échéant, publie certaines données personnelles : nom, prénom (ou pseudo si vous avez fait ce choix dans vos préférences utilisateurs), localisation et date renseignées en saisissant une observation botanique. Ces données sont collectées car :
  • elles sont nécessaires à la définition rigoureuse d’une donnée d’observation (ou donnée d’occurrence de taxon), cf. définition de l’INPN / SINP,
  • elles permettent de rattacher vos observations à votre compte, afin que vous les retrouviez ensuite,
  • elles permettent aux autres membres de vous identifier et vous contacter via la messagerie du site Tela Botanica, afin de vous faire des retours sur l’identification d’une espèce par exemple.

Ainsi, en publiant vos observations et photos sur le site de Tela Botanica vous acceptez de céder :
  • un droit d'usage interne à Tela Botanica, non exclusif, non limité dans le temps et à titre gratuit, sur l’ensemble des données publiques, notamment celui de les détenir, de les dupliquer, de les archiver, de les agréger avec des données provenant d’autres sources et d’effectuer tout type de calcul permettant d’en extraire des informations de niveau général, comme des statistiques par exemple.
  • le droit à Tela Botanica de les diffuser librement et gratuitement sous licence CC-BY-SA 2.0 FR (cf. mentions légales et informations sur les licences) et ce sans limite de temps. Parmi les conditions à cette diffusion, le nom de l'auteur de l'observation ou de la photo ainsi que celui de Tela Botanica doivent être cités. Cependant, dans le cas ou une publication comprendrait plus de 100 auteurs différents, la mention "réseau Tela Botanica" pourra être mentionnée seule.

Vos données d’observations publiques sont notamment susceptibles d'apparaître dans les interfaces (Tela Botanica) suivantes : IdentiPlante, PictoFlora, eFlore, cartographies, téléchargement des données, etc. Dans ce cadre, vos nom et prénom (ou pseudo) apparaîtront comme auteur de l’observation. Dans le cas de l’export, la partie précédent le signe @ de votre adresse électronique sera également affichée. Si des messages vous parviennent, ils auront été envoyés par Tela Botanica via un dispositif garantissant le non piratage de votre adresse.

Enfin, dans le cadre de la valorisation des données issues de sciences participatives, vos données d'observation sont susceptibles d’être transmises au SINP (Système d’Information sur la Nature et les Paysages), pour validation et diffusion.

Vous pouvez à tout moment retirer de la publication les données que vous désirez, mais aussi les consulter, les modifier et les supprimer définitivement depuis votre CEL. Les clauses précédentes ne s'appliqueront plus aux données dépubliées, sans toutefois pouvoir revenir sur les droits cédés lors de leur publication initiale, dans le cas où ces données ont déjà été utilisées par des tiers.
